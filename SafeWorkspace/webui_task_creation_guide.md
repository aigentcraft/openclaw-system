# Claw-Empire Web UI タスク作成ガイド

## 🎯 **目標**
コンサルティング部門に「次世代AIエージェントトレンド調査」タスクを作成

## 🌐 **アクセスURL**
http://127.0.0.1:8800

## 📋 **作成するタスク情報**

### 基本情報
- **タイトル**: 次世代AIエージェントトレンド調査（CEO直接指示）
- **部門**: コンサルティング部門 (dept_consulting)
- **優先度**: 高 (High)
- **タイプ**: 分析 (Analysis)
- **ステータス**: 保留中 (Pending)

### 詳細説明
```
CEO直接指示 (ID: CEO-DIR-20260303-001)

調査目的:
1. コンサルティング部門のクライアント向けにAIエージェント技術の最新動向を把握
2. 業界別の応用可能性と導入事例を収集・分析
3. コンサルティング業務への具体的な活用方法を検討
4. クライアント提案のための材料を整備

調査範囲:
- 技術トレンド: マルチモーダルAI、自律型エージェント、エージェント間協調
- 市場動向: 主要プレイヤー、投資動向、業界別導入状況
- ビジネス応用: コンサルティング業務への適用可能性
- リスク分析: セキュリティ、倫理、規制

期待成果物:
1. 調査報告書（100ページ以上）
2. プレゼンテーション資料（15-20スライド）
3. 実装ガイドライン（導入ロードマップ）
4. ROI分析ツール（スプレッドシート）

スケジュール: 8週間（2026年3月3日〜4月30日）
優先度: 高
部門: コンサルティング部門

指示者: CEO
指示日時: 2026年3月3日
```

## 🛠️ **手順**

### ステップ1: Web UIにアクセス
1. ブラウザで http://127.0.0.1:8800 を開く
2. Claw-Empireのダッシュボードが表示される

### ステップ2: タスク作成画面を開く
1. 左側メニューから「Tasks」または「カンバンボード」を選択
2. 「New Task」または「タスク作成」ボタンをクリック

### ステップ3: タスク情報を入力
1. **Title**: `次世代AIエージェントトレンド調査（CEO直接指示）`
2. **Department**: `Consulting Firm` または `dept_consulting` を選択
3. **Priority**: `High` を選択
4. **Type**: `Analysis` を選択
5. **Description**: 上記の詳細説明をコピー＆ペースト
6. **Due Date**: `2026-04-30` を設定

### ステップ4: タスクを保存
1. 「Create Task」または「保存」ボタンをクリック
2. タスクがコンサルティング部門のインボックスに追加される

## ✅ **確認事項**
1. タスクが正常に作成されたか
2. コンサルティング部門に割り当てられているか
3. 優先度が「高」に設定されているか
4. 期限が2026年4月30日に設定されているか

## 🔍 **トラブルシューティング**
- **Web UIにアクセスできない**: サーバーが起動しているか確認 `curl http://127.0.0.1:8800`
- **部門が表示されない**: 部門ID `dept_consulting` が存在するか確認
- **タスク作成ボタンがない**: ユーザー権限を確認

## 🎉 **成功の目印**
- タスクがコンサルティング部門のカンバンボードに表示される
- タスクIDが発行される
- 進捗状況を追跡できるようになる
